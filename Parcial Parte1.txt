2-las ip se dividen en 5 tipos de clases:
Clase A: esta echa para para servidores de gran capacidad y la cantidad de bits asignados para red son 8 bit y de host 24.
Clase B: esta echa para servidores de tamaño medio y la cantidad de bits asignados son 16 de red y 16 de host.
Clase C: esta echa para servidores de tamaño pequeño y la cantidad de bits asignados para red son 24 bits y 8 bit de host. 
Después existen las Clases D y E pero son experimentales y son utilizados para pruebas.
